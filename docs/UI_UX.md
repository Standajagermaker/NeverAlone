# NeverAlone UI / UX

Design goal: make it obvious that the app helps users meet real people nearby right now.

## Product feeling

- Clean
- Human
- Safe
- Warm
- Fast
- Travel-focused

## Key screens

### 1. Splash

Elements:

- NeverAlone logo
- Slogan: Never travel alone.

### 2. Onboarding

Slides:

1. Meet travelers nearby.
2. Choose what you want to do now.
3. Stay safe with verification and location privacy.
4. Chat with your AI Travel Companion while you wait.

### 3. Login

Options:

- Continue with Apple
- Continue with Google
- Continue with Email

### 4. Profile setup

Required:

- Name
- Age or birth date
- Main photo
- Languages
- Country

Optional:

- Bio
- Interests

### 5. Home / Nearby

Main headline:

> What are you up for?

Activity buttons:

- Coffee
- Beer
- Wine
- Dinner
- Museum
- Beach
- Hiking
- Travel Partner
- Nightlife

After selecting an activity:

- Show people nearby
- Show distance bands
- Show availability expiry

### 6. Map

Shows approximate nearby activity pins, not exact user pins.

### 7. User profile

Sections:

- Photos
- Name and age
- Languages
- Availability
- Bio
- Interests
- Rating
- Report / Block

CTA:

- Invite
- Chat

### 8. Chat

Features:

- Text messages
- Send photo
- Translate
- Report/block shortcut

### 9. AI Companion

Entry points:

- Empty nearby results
- Waiting for reply
- Dedicated tab

Prompt examples:

- What should I do nearby?
- Help me translate this.
- Plan my evening.
- Is this area safe at night?

### 10. Settings

- Profile
- Privacy
- Location
- Languages
- Notifications
- Safety
- Logout

## UX principles

- Never expose exact user coordinates.
- Do not make the app feel like pure dating.
- Make activities expire automatically.
- Encourage real-life safe meeting habits.
- Keep AI supportive but secondary to real people.
