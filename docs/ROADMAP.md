# NeverAlone Roadmap

## Sprint 0 - Foundation

Goal: turn the idea into a structured product and repository.

- Product definition
- Brand direction
- Initial repository
- Core documentation

## Sprint 1 - App foundation

Goal: a runnable mobile/web app shell.

- Choose final frontend framework
- Initialize app project
- Add routing/navigation
- Add design tokens
- Add localization structure
- Add environment config
- Add CI checks

## Sprint 2 - Supabase foundation

Goal: backend ready for auth, profiles and location.

- Create Supabase project
- Add database schema
- Enable Row Level Security
- Add auth providers
- Add storage buckets
- Add local development setup

## Sprint 3 - Profiles and onboarding

Goal: users can sign up and create profiles.

- Onboarding screens
- Email auth
- Profile creation
- Photo upload
- Language and activity preferences

## Sprint 4 - Nearby discovery

Goal: users can see relevant people nearby.

- GPS permission flow
- Approximate location storage
- Radius filters
- Activity status
- Nearby list
- Map view

## Sprint 5 - Chat

Goal: users can talk after mutual interest or accepted invite.

- Conversations
- Realtime messages
- Report/block hooks
- Push notification basics

## Sprint 6 - AI Companion

Goal: AI helps while the user waits or needs travel help.

- AI chat
- Local prompt context
- Translation helper
- Safety and travel suggestions
- Usage limits

## Sprint 7 - Trust and safety

Goal: make the app safer before beta.

- Reporting
- Blocking
- Abuse rules
- Location privacy review
- Moderation queue concept

## Sprint 8 - Beta launch

Goal: first external users.

- TestFlight / internal Android testing
- Analytics
- Crash reporting
- Feedback collection
- App Store / Google Play checklist
